UnfoldFate es una aplicación web sencilla y atractiva para lecturas de tarot que permite a los usuarios seleccionar y revelar una sola carta del tarot desde una cuadrícula barajada. Cada carta revelada muestra su nombre y significado, con una opción para reiniciar y realizar una nueva lectura.

Requisitos funcionales:

- Al principio deben de aparecer todas las cartas boca a bajo.
- El usuario solo puede seleccionar una única carta y al pulsar sobre ella, está se debe de voltear.
- Se debe de mostrar el nombre y la descripción de la misma al voltearla.
- Debe de aparecer un botón para indicar ‘nueva partida’. Al pulsar este botón se deben de poner todas las cartas boca a bajo.
- El orden de las cartas, debe de ser totalmente aleatorio y distinto en cada partida.
